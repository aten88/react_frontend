## Фронтенд проект для тестирования и изучения React
### Как запустить проект:

Клонировать репозиторий и перейти в него в командной строке:

```
git clone https://github.com/aten88/react_frontend
```

```
cd react_frontend
```

Установить зависимости:

```
npm i
```

Запустить проект:

```
npm run start
```
